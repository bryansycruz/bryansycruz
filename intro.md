# IA + AEC: Redefiniendo la Construcción

Este portal explora la intersección entre la **Construcción** y las **Tecnologías de Vanguardia**. Mi propósito es documentar y enseñar la practicidad de la Inteligencia Artificial en nuestra industria.

---

## ¿Qué encontrarás aquí?

**Fundamentos**
: IA y Machine Learning explicados para el ecosistema de la construcción.

**Cloud**
: Despliegue de servicios en AWS y GCP.

**Práctica**
: Notebooks interactivos y recursos abiertos para la comunidad académica.

---

## Propósito

Transformar el conocimiento teórico en herramientas prácticas que optimicen procesos, reduzcan costos y aceleren la innovación en el sector AEC (Architecture, Engineering & Construction).

## ¿Cómo usar este recurso?

1. **Lee** los capítulos en orden o navega por tema.
2. **Clona** el repositorio para ejecutar los notebooks en tu máquina:

   ```bash
   git clone https://github.com/byama/Bryansycruz.git
   cd Bryansycruz
   pip install -r requirements.txt
   ```

3. **Abre** cualquier notebook con Jupyter:

   ```bash
   jupyter notebook
   ```

4. O **lanza directamente en Google Colab** usando el botón 🚀 en cada página.
