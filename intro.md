## Propósito

Transformar el conocimiento teórico en herramientas prácticas que optimicen procesos,
reduzcan costos y aceleren la innovación en el sector AEC (Architecture, Engineering & Construction).

## ¿Cómo usar este recurso?

1. **Lee** los capítulos en orden o navega por tema.
2. **Clona** el repositorio para ejecutar los notebooks en tu máquina:

   ```bash
   git clone https://github.com/bryansycruz/bryansycruz.git
   cd bryansycruz
   pip install -r requirements.txt
   ```

3. **Abre** cualquier notebook con Jupyter:

   ```bash
   jupyter notebook
   ```

4. O **lanza directamente en Google Colab** usando el botón en cada página.
